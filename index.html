  • Étape 1

    Les marqueurs indiquent le nombre de stations présentes. Cliquez sur un marqueur pour effectuer votre réservation.

  • Étape 2

    Une fois que vous avez choisi votre station vous pourrez voir ses informations (adresse, nombre de vélos disponibles, réservation possible)

  • Étape 3

    Dans le panneau "Détails de la station" à droite de la carte, cliquer sur "Réserver" afin d'ouvrir le panneau de réservation.

  • Étape 4

    Pour éffectuer votre réservation, veuillez signer dans le panneau prévu à cet effet. Cliquez sur "Effacer" pour recommencer si besoin, puis cliquez sur "Valider" pour finaliser votre réservation.

  • Étape 5

    Votre réservation est valable 20 min. Cliquez sur "Annuler" pour annuler votre réservation, vous devrez alors recommencer votre réservation.
